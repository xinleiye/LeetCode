# 数字 1 的个数

给定一个整数 n，计算所有小于等于 n 的非负整数中数字 1 出现的个数。

**示例 1：**

``` javascript
输入：n = 13
输出：6
```

**示例 2：**

``` javascript
输入：n = 0
输出：0
```

**提示：**

0 <= n <= 2 * 1e9
