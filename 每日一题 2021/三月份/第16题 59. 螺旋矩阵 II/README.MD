# 螺旋矩阵 II

给你一个正整数 n ，生成一个包含 1 到 n^2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

**示例 1：**

![示例1](./eg1.jpg)

``` javascript
输入：n = 3
输出：[[1,2,3],[8,9,4],[7,6,5]]
```

**示例 2：**

``` javascript
输入：n = 1
输出：[[1]]
```

**提示：**

- 1 <= n <= 20
