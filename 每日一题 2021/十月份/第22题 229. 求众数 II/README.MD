# 求众数 II

给定一个大小为 n 的整数数组，找出其中所有出现超过 `⌊ n/3 ⌋` 次的元素。

**示例 1：**

``` javascript
输入：[3,2,3]
输出：[3]
```

**示例 2：**

``` javascript
输入：nums = [1]
输出：[1]
```

**示例 3：**

``` javascript
输入：[1,1,1,3,3,2,2,2]
输出：[1,2]
```

**提示：**

- `1 <= nums.length <= 5 * 1e4`
- `-1e9 <= nums[i] <= 1e9`
