# LeetCode

一个 HelloWorld 程序猿。

## tools

目录下是一些通用的程序模板

## 双周赛

Leetcode 每隔一周进行的比赛，题目难度不是很稳定。

## 周赛

Leetcode 每周进行的比赛，同双周赛，题目难度也不是很稳定。

## 普通题库

保存平时随机刷的题，按照题目的主要解法分类。

## 每日一题

LeetCode 官方的打卡活动，目前（2020年）从 **3** 月到 **5** 月，每天出一道题，**ac**后，即表示打卡成功。题目会与 **普通题库** 重复

## 名词说明：

- ac: accept，通过了所有测试用例。
