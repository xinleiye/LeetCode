# 翻转二叉树

翻转一棵二叉树。

**示例：**

``` javascript
输入：

     4
   /   \
  2     7
 / \   / \
1   3 6   9
输出：

     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
