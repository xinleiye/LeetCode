# 移除重复节点

编写代码，移除未排序链表中的重复节点。保留最开始出现的节点。

**示例1:**

``` javascript
 输入：[1, 2, 3, 3, 2, 1]
 输出：[1, 2, 3]
```

**示例2:**

``` javascript
 输入：[1, 1, 1, 1, 2]
 输出：[1, 2]
```

**提示：**

- 链表长度在[0, 20000]范围内。
- 链表元素在[0, 20000]范围内。
